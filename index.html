Principal

El objetivo de este blog es el de servirme como hoja de pruebas para prácticar distintas habilidades HTML y CSS que vaya aprendiendo. De paso cañaso, lo uso medio de libreta y puedo anotar recetas, guardar algunas fotos/recuerdos, jugar con formularios, etc. No te tomes demasiado encerio nada que te encuentres acá, excepto las recetas, esas si tomatelas encerio, o se te quema la casa.
